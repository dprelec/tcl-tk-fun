Fun with Tcl/Tk
===============

Doing fun stuff with Tcl/Tk, simple experiments in drawing.

## bifurcation_diagram.tcl

Draws bifurcation diagram for logistic map, see wikipedia:
http://en.wikipedia.org/wiki/Bifurcation_diagram

## henon_map.tcl

Draws Hénon map, see wikipedia: http://en.wikipedia.org/wiki/H%C3%A9non_map  

## sierpinski_carpet.tcl

Draws Sierpinski carpet, see wikipedia: http://en.wikipedia.org/wiki/Sierpinski_carpet
